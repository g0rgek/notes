Наследования нет, есть [[struct#Встраивание типов]] в другие структуры.
Чтобы навесить на zap свои методы, нужно обернуть изначальной объект zap в структуру, она унаследует все методы zap, и мы можем навесить своих:
```go
import "zap"

type Logger struct{
	zap.Logger
	severity string
}

func main(){
	zap := zap.NewLogger()
	l := &Logger{zap, "INFO"}
	l.Logger.Log() //обращаемся к Zap-логеру
}
```
