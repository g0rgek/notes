- Возвращает `uintptr` — **смещение (в байтах)** поля внутри содержащей структуры.
- Операнд — выражение вида `s.field` (поле struct-переменной).
- Результат — константа времени компиляции.
- Полезно для проверки/вычисления [struct#Padding](struct#Padding.md) и ручной упаковки структур.    

```go
var x struct { 
	A byte
	// _[3]byte
	B int32 
} 
fmt.Println(unsafe.Offsetof(x.B)) // обычно 4 (из-за выравнивания)
```
