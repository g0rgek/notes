- Возвращает `uintptr` — **требуемое выравнивание (в байтах)** для типа выражения.
- Аргумент может быть любым выражением; возвращает выравнивание его типа.
- Помогает понять, почему компилятор вставляет [struct#Padding](struct#Padding.md) между полями.

```go
var x int32 
fmt.Println(unsafe.Alignof(x)) // обычно 4 на 32/64-битных архитектурах
```
